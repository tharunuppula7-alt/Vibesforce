### Deployment Steps

1. **Deploy Apex Classes**
   ```bash
   sf project deploy start --metadata ApexClass:DoctorSearchService
   sf project deploy start --metadata ApexClass:PatientIdentificationService
   sf project deploy start --metadata ApexClass:AppointmentBookingService
   sf project deploy start --metadata ApexClass:AppointmentRetrievalService
   sf project deploy start --metadata ApexClass:AppointmentCancellationService
   sf project deploy start --metadata ApexClass:FAQSearchService
   sf project deploy start --metadata ApexClass:ConversationLogger
   ```

2. **Validate Agent Compilation**
   ```bash
   sf agent validate authoring-bundle --json --api-name OncoGlobal_Patient_Scheduling_Agent
   ```

3. **Preview with Live Actions**
   ```bash
   sf agent preview start --json --use-live-actions --authoring-bundle OncoGlobal_Patient_Scheduling_Agent
   ```

4. **Test with Sample Utterances**
   - "I need to book an appointment with a cardiologist"
   - "Show me my upcoming appointments"
   - "Cancel my appointment with confirmation code ABC123"
   - "What are your office hours?"

5. **Publish and Activate**
   ```bash
   sf agent publish authoring-bundle --json --api-name OncoGlobal_Patient_Scheduling_Agent
   sf agent activate --json --api-name OncoGlobal_Patient_Scheduling_Agent
   ```
